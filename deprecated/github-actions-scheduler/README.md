## Deprecated GitHub Actions Scheduler

This folder archives the previous GitHub Actions based scheduler implementation.

Dify Cloud now supports scheduler triggers directly inside Workflow Studio, so this code is no longer used by the active project. It is kept here only for historical reference.

Archived contents:

- `.github/workflows/dify-scheduler.yml`: previous GitHub Actions workflow
- `scheduler/`: Node.js scheduler implementation and local test runner
- `package.json` / `package-lock.json`: dependencies and scripts for the archived scheduler
